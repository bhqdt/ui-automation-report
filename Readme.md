# 基于LLM的移动端UI自动化 - 中期汇报网页

## 1. 项目简介

本项目旨在展示“基于LLM的UI自动化”中期汇报内容。网页详细介绍了项目背景、技术路线（UI感知、语义理解、动作规划）、当前进展以及下一步计划。

## 2. 文件结构

- `index.html`: 网页的主文件，包含了所有的汇报内容和排版样式。
- `README.md`: 本使用手册，说明如何查看网页。

## 3. 如何查看网页

### 方法一：在线预览（推荐）

本项目已部署在GitHub Pages，请访问以下链接直接查看：
https://bhqdt.github.io/ui-automation-report/

### 方法二：本地查看

1. 下载 `index.html` 文件到您的电脑。
2. 双击文件，它将自动在您的默认浏览器（如Chrome, Edge）中打开。

## 4. 技术说明

- 网页使用 **HTML5** 构建结构。
- 使用 **Tailwind CSS** (CDN版本) 进行快速样式美化，无需安装Node.js或其他依赖，开箱即用。
